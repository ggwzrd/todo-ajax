## Get Started

1. Install [RVM](https://rvm.io/rvm/install)
2. Install [Rails](http://installrails.com/steps/choose_os)
3. Copy content from [this repository](https://gist.github.com/hezus/1834978f7aa8a576d866e59fd0ad7c72)
4. ``` rails db:drop ```
5. ``` rails db:create ```
6. ``` rails db:migrate ```
7. ``` rails db:seed ```
